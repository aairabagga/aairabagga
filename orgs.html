Illinois Business Consulting

Once COVID-19 hit the US, many local businesses has to close down, including International Diamond Company. I built this e-commerce website for a local jewelry store in my area to allow for an online inventory and online purchases. This website is optimized for mobile and tablet devices of all sizes as most e-commerce traffic comes from mobile devices. For payment processing, I used the Stripe API which allows for international payments and two-factor authentication. Aditionally, I implemented Strapi, a headless CMS, which allows for easy database manipulation and creates an admin page, with the power to create new data types and add inventory.
